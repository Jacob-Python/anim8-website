Anim8

How-tos for Anim8

+ Basics

Anim8 is a simple, easy, basic animation and design app for beginners.
It's a great way to get kids and beginners into art and animation. It's made with Java.



+ Advanced topics

 + The Anim8 Markup Standard

 Anim8 uses a form of markup language called Animarkup.
 The language is rather simple if you've learned HTML or XML. A tag is a block of code that holds data or other tags. Whole tags are opened with @foo and closed with @end. However,
 individual pieces of data (partial tags) are formatted like this: @foo bar baz qux
 Some examples of tags are tags that store project names, animations, and shapes.

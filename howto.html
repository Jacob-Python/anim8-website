+ About Anim8

Anim8 is a simple, easy, basic animation and design app for beginners.
It's a great way to get kids and beginners into art and animation. It's made with Java.



+ How it Works

Anim8 is made with Java. Anim8 uses Processing and Swing for UI and controls.
Anim8 uses Xuggle-Xuggler to make videos.
Files are stored with the Anim8 Markup Standard in the .a8p file type.
